# Generate Switch Statistics
### Example usage

```
./generate_stats.py example_statistics.txt \
--total-ports 48 \
--uplink1 15 \
--uplink2 17  \
--unicast 0 \
--unicast-max 1 \
--broadcast 1 \
--broadcast-max 5 \
--broadcast-limit 10024 \
--packet-size -1 \
--multicast 1 \
--multicast-max 3 \
--multicast-limit 1 \
--loop 2 \
--loop-interface1 -1 \
--loop-interface2 -1 \
--vlan 100 \
--runtime 11 \
--loop-after 0
```
